# TFM elasticidades leche UHT

Este repositorio contiene la base de datos depurada utilizada como material de apoyo para el Trabajo Fin de Máster titulado:

**Elasticidad-precio de la demanda de leche UHT en el mercado español: dinámica y heterogeneidad por marca**

El estudio utiliza datos semanales para el mercado español de gran distribución, con información de ventas, precios, presencia en folleto y variables auxiliares empleadas en el análisis empírico.

## Contenido

La carpeta `datos/` incluye:

- `base_tfm_clasicas.csv`: base principal del segmento de leches clásicas.
- `base_tfm_sin_lactosa.csv`: base utilizada para la réplica en el segmento de leche sin lactosa.
- `diccionario_variables.csv`: descripción de las principales variables incluidas en las bases.
- `precios_leche_mensual.csv`: serie mensual del precio de la leche cruda.
- `precio_electricidad_mensual.csv`: serie mensual del precio de la electricidad.
- `precio_combustible_mensual.csv`: serie mensual del precio del combustible.
- `ipc_leche_mensual.csv`: serie mensual del IPC de leche.

## Uso de los datos

Los datos se facilitan con fines académicos y de reproducibilidad del TFM. La información ha sido estructurada para permitir la comprensión de la base empírica utilizada en el análisis.
